Hello, thanks for taking a look at my website, I had tons of fun playing around to make my website where it is now.

All files for my webiste are located in the WEBSCRIPTA32022
it consists of the following folders:

> Node_modules
    -(Contains all the dependencies)
> public
    -Assets (contains icons)
    -Content (contains css file)
    -Scripts (contains javascript)
> server
    -config (contains app.js & db.js)
    -controller (contains airplane.js and index.js)
    -models (contains airplane.js)
    -routes (contains 3 routes: index, airplane & users)
> views
    -airplane (contains the view for displaying add, edit & list in .ejs format)
    -partials (contains the shared header, footer and Banner)
    -error.ejs (view for displaying errors)
    -index.ejs (view for displaying homepage)
